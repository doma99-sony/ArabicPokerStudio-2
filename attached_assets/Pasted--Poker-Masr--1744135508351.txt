🎮 مشروع لعبة "بوكر مصر" - Poker Masr 🎮
مرحبًا،
دلوقتي هنبدأ نشتغل على لعبة جديدة ضمن منصة الألعاب باسم "بوكر مصر"، وهي لعبة بوكر أونلاين متعددة اللاعبين، حية (Real-time)، ومتصلة بمنصتنا من حيث الحسابات، الرصيد، والمكافآت.

✅ المهام المطلوبة بالتفصيل:
إنشاء مجلد خاص باللعبة

اسم المجلد: poker-masr

المسار: client/src/pages/poker-lobby/poker-masr

بعد إنشاء المجلد، ابعتلي عشان نبدأ نضيف التفاصيل الباقية من تصميم وواجهة ومنطق اللعبة.

🔧 لغات البرمجة والتقنيات المطلوبة:
👨‍💻 Frontend (الواجهة الأمامية):
React.js: لبناء الواجهة بشكل ديناميكي وسريع

TypeScript: لضبط نوع البيانات وتقليل الأخطاء

Socket.IO Client: للتواصل الفوري مع السيرفر

CSS / SCSS: لتصميم واجهة جذابة وسلسة

Howler.js: لتشغيل المؤثرات الصوتية

🧠 Backend (الخادم ومنطق اللعبة):
Node.js + Express: لبناء REST API والتحكم في اللعبة

Socket.IO Server: لتحديثات فورية بين جميع اللاعبين

Game Logic Engine: كود مخصص يدير منطق البوكر (توزيع الورق، ترتيب الأدوار، الرهانات)

💾 Database (قاعدة البيانات):
MongoDB أو PostgreSQL: لتخزين بيانات اللاعبين، نتائج الجولات، الرصيد والمكافآت

Redis: لتخزين الجلسات المؤقتة وتسهيل التواصل بين اللاعبين

🔐 Authentication (تسجيل الدخول والربط):
Firebase Auth أو JWT Auth: لربط اللاعب بحسابه على المنصة وربط الرصيد

🎮 مشروع لعبة سلوتس – "كتاب الفرعون" (Pharaoh’s Book)
🧠 فكرة اللعبة:
نريد تطوير لعبة سلوتس (slots) متكاملة، مستوحاة من لعبة Book of Dead، لكنها تحمل اسمنا الخاص:
📕 "كتاب الفرعون" (Pharaoh’s Book)
اللعبة ستكون جزءًا من منصتنا الخاصة، ويجب أن يتم ربطها بشكل كامل بالمنصة، بنظام الرقائق (tokens/coins)، ونظام المستخدمين، مع واجهة احترافية وجاذبة بصريًا.

🎯 الهدف:
نسخة "كوبي بيست" (مطابقة قدر الإمكان) للعبة Book of Dead، تشتغل على منصة Replit وتكون قابلة للتوسيع مستقبلاً.

🔗 رابط اللعبة الأصلية للمشاهدة والرجوع إليها كمصدر:
https://www.playngo.com/games/rich-wilde-and-the-book-of-dead

🛠 التقنيات المطلوبة:
✅ الواجهة (Frontend):
HTML + CSS + JavaScript

استخدام Canvas أو DOM حسب الحاجة

تصميم شاشة رئيسية، زر “Spin”، عرض الرصيد، عرض الرموز

واجهة مكونة من:

5 أعمدة × 3 صفوف لعرض الرموز

زر Spin

عرض عدد الرقائق (Coins)

إظهار نتيجة اللفة (Win / Lose)

مؤثرات حركة (Animation) عند الربح أو المكافأة

صوت عند الضغط على Spin، الربح، أو ظهور رمز خاص (زي الكتاب)

✅ السيرفر (Backend):
Python (يفضل Flask)

إدارة جلسة اللعب

ربط الرقائق بحساب اللاعب

التحكم في المكافآت (Free Spins، أرباح، إلخ)

حساب النتائج عشوائيًا باستخدام RNG (Random Number Generator)

API endpoints عشان المنصة تقدر:

تبدأ جولة جديدة

تخصم أو تضيف رقائق

تخزن النتائج

ترجع بيانات اللاعب

✅ الربط مع المنصة:
استخدام Token أو معرف اللاعب من المنصة

عند بداية اللعبة يتم جلب رصيد اللاعب من المنصة

عند كل جولة يتم خصم الرقائق أو إضافتها حسب النتيجة

حفظ النتائج داخل قاعدة بيانات المنصة أو إرسالها عبر API

اللعبة تفتح داخل iframe أو صفحة خاصة داخل المنصة

✅ العناصر الفنية:
صور رموز مستوحاة من Book of Dead:

رمز البطل الفرعوني (شبيه Rich Wilde)

الكتاب الذهبي

تمثال، جعران، عين حورس، إلخ

صور يجب أن تكون شفافة (PNG)

يمكن استخدام رموز مؤقتة (placeholder) لو الصور غير متوفرة، ويتم استبدالها لاحقًا

تأثيرات صوتية:

عند الضغط على زر Spin

عند الربح

عند تفعيل Free Spins

✅ المطلوب تنفيذه:
تصميم واجهة مطابقة للعبة Book of Dead

تطوير لعبة سلوتس من 5 أعمدة × 3 صفوف

تحريك الرموز عند كل "Spin"

حساب النتيجة عشوائيًا مع إمكانية وجود “Free Spins”

عرض رصيد اللاعب وربطه بمنصة خارجية (token/session)

خصم أو إضافة الرقائق تلقائيًا

تصميم تأثيرات صوتية وبصرية

كتابة كود نظيف، مقسم، سهل التوسيع

أي سؤال أو نقص في الصور أو الأصوات، ممكن يتم استخدام عناصر مؤقتة ريثما يتم توفير النسخة الأصلية أو المعاد تصميمها.

بالتوفيق، والمنتظر أول نسخة شغالة على Replit ✨

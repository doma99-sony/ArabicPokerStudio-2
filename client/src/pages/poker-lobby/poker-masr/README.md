# بوكر مصر (Poker Masr)

هذا المجلد يحتوي على مكونات وصفحات خاصة بلعبة بوكر مصر ضمن تطبيق الألعاب الشامل.

## محتويات المجلد:

- `index.tsx`: الصفحة الرئيسية للعبة بوكر مصر التي تعرض غرف اللعب المختلفة
- المكونات الفرعية للعبة ستضاف قريباً

## أنواع الغرف:

1. بوكر تكساس هولديم
2. مكة بوكر
3. بوكر تركي كلاسيك
